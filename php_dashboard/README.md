# Dubai Property Management PHP Dashboard

## How to Deploy

1. **Upload the Folder**
   - Upload the entire `php_dashboard` folder to your `public_html` directory on Hostinger (or any PHP web host).
   - You can rename the folder if you want (e.g., to `dashboard`).

2. **Access the Dashboard**
   - Visit `https://yourdomain.com/php_dashboard/` (or the folder name you used).
   - The main entry point is `index.php`.

3. **Requirements**
   - PHP 7.2 or higher (no database required for mock/demo data)
   - Internet access for Bootstrap CDN

4. **Customization**
   - All main pages are in the `pages/` folder.
   - Styles are in `assets/style.css`.
   - You can add real data or connect to a database as needed.

---

**This is a static demo dashboard. For real data, connect to a database and update the PHP logic.**
